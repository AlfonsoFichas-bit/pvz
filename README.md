# Plants vs. Zombies - Rework en Godot

Un reimplemetación basada en Godot del clásico juego de defensa tipo torre Plants vs. Zombies.

## Visión General

Este proyecto es una recreación completa del popular juego Plants vs. Zombies usando el motor Godot. El objetivo es recrear las mecánicas de juego principales, niveles y características manteniendo la sensación nostálgica del juego original.

## Características

- Jugabilidad clásica de Plants vs. Zombies
- Múltiples tipos de plantas con habilidades únicas
- Varios enemigos zombies con diferentes comportamientos
- Mecánica de recolección de sol
- Sistema de progresión por niveles

## Cómo Empezar

1. Clona el repositorio:
   ```bash
   git clone [url-del-repositorio]
   ```

2. Abre el proyecto en Godot Engine:
   - Inicia el Editor de Godot
   - Importa el directorio del proyecto
   - Abre el proyecto

3. Ejecuta el juego:
   - Haz clic en el botón "Jugar" en el Editor de Godot
   - O exporta a tu plataforma deseada
